# Textile Armour
# Version: 2026-03-16
# Compilation of all Textile (Padded/Organic) armor rules, tables, and extensions for Calradia RP. This tier focuses on soft, flexible protection.
# Integrates with the Universal Armour Upgrade Components Table (see below). Upgrades are applied based on material composition (padded/textile/soft leather).
# Global Layering: Textile as inner layer adds 50% AC (round down) to outer tiers; +20% total Enc. When outer succeeds coverage, inner auto-applies full AC.
# Craftsmanship: Common base; Good (+1 AC all, x3 cost); Best (+2 AC, x10 cost).
# Unveiling Ties: Mutant hides +1 AC but +10% mutation risk on False Moons.
# Coverage Rule: Coverage only ever refers to Head, Torso, Legs and Arms.

## Textile Torso Interpretation Rules
For Textile torso armor pieces, parse the item title for type and coverage keywords to determine base AC (Cut/Pierce/Blunt), Enc, and coverage percentages. All Textile torso items provide 100% Torso Coverage by default. Coverage keywords modify Arm and Leg protection:
- Arm Coverage: Sleeveless (0%), Short Sleeved (30%), Long Sleeved (90%). If unspecified, assume Short Sleeved (30%).
- Leg Coverage: Short (0%), Long (40%). If unspecified, assume 20% (standard skirt/hem length; do not label as "Default" in title).
AC applies proportionally to covered areas. Uncovered areas = 0 AC. For partial coverage, roll d100 ≤ coverage % on hit to apply AC; failure = no protection.
Base Enc from type; increase Enc by +2.5 for each raised coverage. Round Enc to nearest 5.
Upgrades from the Universal Armour Upgrade Components Table may be applied if the material matches (padded/textile).
Situational modifiers in Notes.
Craftsmanship applies as global (+1/+2 AC); inherent flex for +10 Dodge if Enc <50.
Example Parsing: "Long Sleeved Gambeson with High-Density Quilting" → Gambeson base (3/1/3, 10 Enc) + Long Sleeved (+2.5 Enc) + High-Density Quilting (+1/+1/+1 AC, +3 Enc) = Total Enc 15.5 (round 15); 100% Torso, 90% Arms, 0% Legs.

## Textile Leg Armour Interpretation Rules
For Textile Leg Armour pieces, parse the item title for type and coverage keywords to determine base AC (Cut/Pierce/Blunt), Enc, and coverage percentages. Types from Textile torso apply as prior, with base AC/Enc scaled down 50% for legs only.  
Coverage keywords modify Leg protection percentage.  
AC applies proportionally to covered leg areas. Uncovered = 0 AC. Roll d100 ≤ coverage % on leg hit for AC.
Base Enc from type (50% of torso base, round up); increase Enc by +2.5 for raised coverage.
Upgrades from the Universal Armour Upgrade Components Table may be applied if the material matches (padded/textile).
Craftsmanship global (+1/+2 AC); inherent flex +5 Dodge if Enc <30.
Example Parsing: "Padded Chausses with River Troll Hide" → Padded Chausses base (2/1/2, 6 Enc) + River Troll Hide (+1/+1/+1 AC, +5 Enc) = Total Enc 11; 80% Legs.

## Textile Head Protection
# Interpretation Rules for AI
For Textile head protection pieces, parse the item title for type to determine base AC (Cut/Pierce/Blunt), Enc, and coverage percentages. All coverage applies to the head in its entirety (skull and scalp). Padded Coif/Hood is treated as 75% Head Coverage by default.
AC applies to the full head area. For partial coverage items, roll d100 ≤ coverage % on head hit to apply AC; failure = no protection.
Base Enc from type; increase Enc by +2.5 for raised coverage or added padding. Round Enc to nearest 5.
Upgrades from the Universal Armour Upgrade Components Table may be applied if the material matches (padded/textile).
Craftsmanship global (+1/+2 AC); no Qualities—use inherent flex for +10 Dodge if Enc <20.
