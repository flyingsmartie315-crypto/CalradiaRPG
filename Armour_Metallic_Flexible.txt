# Metallic Flexible Armour (Mail & Scale Tier)
# Version: 2026-03-16
# Flexible metallic tier: riveted mail, butted mail, scale, and related pieces. Superior cut/pierce resistance, weaker vs. blunt. Excellent layering base under rigid tiers.
# Integrates with the Universal Armour Upgrade Components Table (applicable based on material composition).
# Global Layering: Adds full AC when outer; inner Textile auto-applies 50% if outer succeeds coverage.
# Craftsmanship: Common base; Good (+1 AC all, x3 cost); Best (+2 AC, x10 cost).
# Unveiling Ties: False Moons cause rust/corrosion (-1 AC temporary on metal); scale from Unveiling beasts +1 AC but +5% mutation risk.
# Coverage Rule: Coverage only ever refers to Head, Torso, Legs and Arms.

## Metallic Flexible Torso Interpretation Rules
For Metallic Flexible torso armor pieces, parse the item title for type and coverage keywords to determine base AC (Cut/Pierce/Blunt), Enc, and coverage percentages. All torso items provide 100% Torso Coverage by default unless specified otherwise. Coverage keywords modify Arm and Leg protection:
- Arm Coverage: Sleeveless (0%), Short Sleeved (30%), Long Sleeved (90%). If unspecified, assume Short Sleeved (30%).
- Leg Coverage: Short (0%), Long (40%). If unspecified, assume 20%.
AC applies proportionally to covered areas. Uncovered areas = 0 AC. For partial coverage or gaps, roll d100 ≤ coverage % on hit to apply AC; failure = no protection.
Base Enc from type; increase Enc by +3 for each raised coverage. Round Enc to nearest 5.
Upgrades from the Universal Armour Upgrade Components Table may be applied if the material matches.
Situational modifiers in Notes.
Craftsmanship applies as global (+1/+2 AC); inherent flex for +5 Dodge if Enc <80.
Example Parsing: "Long Sleeved Riveted Mail Hauberk with Mail Voiders" → Riveted Mail Hauberk base (4/4/4, 70 Enc) + Long Sleeved (+3 Enc) + Mail Voiders (20 Enc, gap-only AC) = Total Enc 93 (round 95); 100% Torso, 90% Arms (voiders add AC to armpit gaps).
