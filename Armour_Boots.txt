# Boots Armour
# Version: 2026-03-16
# Standalone modular boot system for Calradia RP. Boots extend Leg Armour and can use materials from Textile, Metallic Flexible, or Rigid Partial/Rigid Uniform tiers. They are fully compatible with the Universal Armour Upgrade Components Table.
# Coverage Rule: Boots always provide 100% Foot Coverage + variable Leg Coverage based on height/material. Coverage only ever refers to Legs (feet subsumed).
# Craftsmanship: Common base; Good (+1 AC all, x3 cost); Best (+2 AC, x10 cost).
# Upgrades: Use the Universal Armour Upgrade Components Table. Applicable upgrades are determined by material composition.
# Location Tags: Added in Notes column for all modules and materials.

## Global Layering & Tier Classification
Boots are modular and their tier is determined by primary material:
- Textile materials (Padded, Leather, Suede, Felt & Leather Hybrid) → treated as Textile tier.
- Rigid materials (Cuir Bouilli, Splinting) → treated as Rigid Partial tier.
- Greaves → treated as Rigid Uniform tier.

Boots can layer over Textile Leg Armour or under Rigid greaves. When outer succeeds coverage, inner auto-applies 50% AC.

## Boot Interpretation Rules
For any boot, parse the item title for type, material, height, and upgrade keywords to determine base AC (Cut/Pierce/Blunt), Enc, and coverage percentages.  
- Height Keywords: Ankle (15% Leg Coverage), Mid-Calf (30%), Knee-High (50%). If unspecified, assume Ankle (15%).  
- AC applies proportionally to covered leg areas (feet always 100%). Uncovered = 0 AC. Roll d100 ≤ coverage % on leg/foot hit for AC.  
- Base Enc from type + material; increase Enc by +2.5 for raised height or applicable upgrades. Round Enc to nearest 5.  
- Upgrades from the Universal Armour Upgrade Components Table may be applied if the material matches.  
- Craftsmanship global (+1/+2 AC).
